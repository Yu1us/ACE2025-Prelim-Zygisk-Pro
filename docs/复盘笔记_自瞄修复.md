# 复盘笔记：自瞄修复

**问题**: 发射子弹时，枪口自动锁定固定白色箱子。

## 1. 问题分析

### 1.1 现象

- 射击后摄像机强制对准特定白色箱子
- 移动视角无效，射击时被拉回
- 比赛故意设计的异常

### 1.2 锁定目标

逐个摧毁箱子排除法测试，锁定目标:

```
EditorCube8
```

## 3. IDA 静态分析

### 3.1 目标函数

函数入口：`sub_670F110`

### 3.2 关键代码逻辑

```c
// 读取 EditorCube8 的 RootComponent
LDR     X8, [X28,#0x130]        ; RootComponent

// 读取坐标 RelativeLocation (offset 0x1D0)
LDR     S0, [X9,#0x1D0]         ; X
LDR     S2, [X9,#0x1D4]         ; Y
LDR     S3, [X9,#0x1D8]         ; Z

// ... 计算目标方向的旋转值 ...

// 读取 vtable 并调用 SetControlRotation
LDR     X8, [X0]                ; 读取 vtable
LDR     X8, [X8, #0x698]        ; vtable offset 1688 = SetControlRotation
BLR     X8                      ; ★ 强制设置视角旋转
```

### 3.3 vtable 偏移说明

| 偏移 | 十进制 | 函数名 |
|------|--------|--------|
| `0x698` | 1688 | `SetControlRotation` |

### 3.4 补丁位置

| 地址 | 指令 | 字节码 |
|------|------|--------|
| `0x670F3F8` | `BLR X8` | `00 01 3F D6` |

---

## 4. 修复方案

### 4.1 补丁方法

将 `BLR X8` 指令替换为 `NOP`，阻止强制旋转调用。

| 补丁地址 | 原指令 | 补丁后 |
|----------|--------|--------|
| `0x670F3F8` | `BLR X8` (`00 01 3F D6`) | `NOP` (`1F 20 03 D5`) |

### 4.2 Frida 验证脚本

```javascript
// scripts/solve_aim.js
const libBase = Module.findBaseAddress("libUE4.so");
const patchAddr = libBase.add(0x670F3F8);

// 读取原始字节
console.log("[*] Original bytes:", hexdump(patchAddr, { length: 4 }));

// 修改内存权限并打补丁
Memory.protect(patchAddr, 4, 'rwx');
patchAddr.writeByteArray([0x1F, 0x20, 0x03, 0xD5]);  // ARM64 NOP

console.log("[+] Patched! Aimbot disabled.");
```

### 4.3 C++ 实现（Zygisk）

```cpp
// hacks/mod_aimfix.cpp

#include "game_offsets.h"
#include "memory_utils.h"

namespace Game {
    // 自瞄强制旋转调用位置
    constexpr uintptr_t AimLock_SetControlRotation = 0x670F3F8;
}

void fix_aimbot(uintptr_t libBase) {
    uintptr_t patchAddr = libBase + Game::AimLock_SetControlRotation;
    
    // ARM64 NOP: 0xD503201F (little-endian: 1F 20 03 D5)
    static const uint8_t nop[] = { 0x1F, 0x20, 0x03, 0xD5 };
    
    /* 🛑 填空题: 写入 NOP 指令到 patchAddr
     * 提示: 
     *   1. 需要先修改内存页权限为可写 (mprotect)
     *   2. 然后用 memcpy 写入 nop 数组
     *   3. 最后恢复内存权限
     * 参考: mod_gunoffset.cpp 中的内存写入方式
     */
}
```
